# 📊 Customer Churn Prediction System
# 📌 Project Overview
Customer churn is one of the most important business metrics for subscription-based and e-commerce companies. This project uses Machine Learning to predict whether a customer is likely to leave (churn) or stay with the company based on customer behavior, purchasing patterns, and subscription details.

### 🚀 Live Demo
[Open Live Application](https://customer-churn-predictions-1-7z2w.onrender.com)

The application is built using:
* Python
* Flask
* Scikit-Learn
* Gaussian Naive Bayes
* HTML/CSS
* Render Cloud Platform
---

# 🎯 Problem Statement

Businesses lose revenue when customers stop using their services. Predicting churn helps organizations:

* Improve customer retention
* Reduce customer acquisition costs
* Increase customer satisfaction
* Identify high-risk customers
* Design targeted marketing campaigns
---

# 📂 Dataset Information

Dataset: E-Commerce Customer Churn Dataset

The dataset contains approximately 200,000 customer records with various customer attributes and behavioral metrics.

### Features Used

| Feature                | Description                    |
| ---------------------- | ------------------------------ |
| customer_id            | Unique Customer ID             |
| age                    | Customer Age                   |
| gender                 | Gender                         |
| city                   | Customer City                  |
| tenure_months          | Customer Relationship Duration |
| avg_order_value        | Average Order Value            |
| total_orders           | Total Orders                   |
| last_purchase_days_ago | Days Since Last Purchase       |
| support_tickets        | Customer Support Tickets       |
| subscription_type      | Subscription Plan              |

### Target Variable
* Churn (0 = Stay, 1 = Churn)

---

# 🧠 Machine Learning Model

### Algorithm Used
Gaussian Naive Bayes (GaussianNB)

### Why GaussianNB?
* Fast Training
* Lightweight Model
* Works Well for Classification
* Good Baseline Model
* Suitable for Large Datasets
